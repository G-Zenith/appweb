# Ticket: Rediseño visual radical del panel administrativo

## Estado

- Prioridad: Alta
- Tipo: Diseño / Frontend
- Locale principal: Español
- Flujo de trabajo: TDD con ciclos red/green
- Repositorio: Inicializado localmente

## Objetivo

Transformar la interfaz actual en una experiencia mucho más moderna, marcada y diferenciada del estado inicial, manteniendo la aplicación funcional y clara para tareas administrativas.

## Alcance inicial

- Rediseñar el shell principal de la app.
- Replantear el dashboard con una dirección visual más fuerte.
- Mejorar jerarquía visual, tipografía, color, espaciado y composición.
- Hacer que tablas y paneles se sientan más actuales y coherentes.
- Mantener la interfaz responsive.

## Criterios de aceptación

- La interfaz debe verse claramente distinta a la versión actual.
- Debe existir una dirección visual consistente y moderna.
- El dashboard debe incluir una cabecera más editorial y secciones con mejor priorización.
- La navegación lateral y el contenedor principal deben sentirse más refinados.
- Los cambios principales deben cubrirse con pruebas creadas antes de la implementación.
- El trabajo debe avanzar en pasos pequeños con commits frecuentes.

## Plan de ejecución

1. Crear pruebas para la nueva estructura visual principal.
2. Implementar el rediseño del layout y dashboard hasta pasar las pruebas.
3. Extender el nuevo lenguaje visual al resto de páginas.
4. Validar con lint y pruebas.
5. Registrar commits por cada hito importante.

## Notas

- La app ya está en español.
